# Wikia Release Readiness Group Chat

Use this group chat after lane branches have completed focused tests and before integration release.

```text
all lane agents
   |
   v
root moderator
   |
   v
release decision
```

Prompt:

```text
Review all Wikia lane outputs and identify integration risks. Each lane must summarize changed paths, tests run, known risks, and rollback. Moderator outputs merge order, blocked items, and whether release integration may start. Do not deploy from the group chat.
```
